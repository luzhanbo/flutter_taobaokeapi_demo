# 淘宝客API演示【flutter版】

淘宝客API演示【flutter版】支持 android/ios/web.

## 依赖组件
- flutter_taobaokeapi
- flutter_taobaokeui

## 测试应用
1. 安装Flutter环境，参考：https://flutter.cn/docs/get-started/install
2. flutter channel master
3. flutter upgrade
4. flutter pub get
5. flutter run
